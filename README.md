# Blog_WebApp

A dynamic Blog Web Application developed using PHP, MySQL, AJAX, and DataTables, following the MVC architecture for scalable and maintainable development. The project provides secure authentication, blog management features, and responsive user interaction.

---

## 🚀 Features

* User Authentication System

  * Secure Login & Signup
  * Session Management
  * Form Validation

* Blog Management

  * Create Blog Posts
  * Read/View Posts
  * Update Existing Posts
  * Delete Posts

* AJAX Integration

  * Dynamic content updates without page reload

* DataTables Integration

  * Pagination
  * Sorting
  * Search functionality

* Responsive User Interface

## 📥 Installation Guide

### Step 1: Download and Install XAMPP

Download the latest version of XAMPP from the official website:

https://www.apachefriends.org/

Install XAMPP in the `C:` drive.

---

### Step 2: Move Project to htdocs

Go to:

```bash
C:\xampp\htdocs
```

Create a folder named:

```bash
Blog_WebApp
```

---

### Step 3: Clone the Repository

Open Git Bash or Command Prompt inside the `htdocs` folder and run:

```bash
git clone https://github.com/ayushyadav114/Blog_WebApp.git
```

---

### Step 4: Start Apache and MySQL

Open the XAMPP Control Panel and start:

* Apache
* MySQL
---

### Step 5: Import Database

1. Open your browser and go to:

```bash
http://localhost/phpmyadmin
```

2. Create a new database.

3. Import the SQL file provided in the repository.

---

### Step 6: Run the Project

Open any browser and type:

```bash
http://localhost/Blog_WebApp
```
